# DigitalBuku Assessment Engine

A production‑ready assessment engine for generating reports (Leadership, DISC, Values, Competency, 360°, Psychometric) from PDF and Excel inputs.

## Folder Structure

```
DigitalBuku-Assessment-Engine/
├─ .gitignore
├─ LICENSE
├─ README.md
├─ package.json
├─ assets/            # Static assets (images, fonts, etc.)
├─ components/        # UI components – to be implemented later
├─ config/            # Configuration files (env, settings)
├─ docs/              # Documentation and design specs
├─ input/             # Sample input files (PDFs, Excel)
├─ output/            # Generated output (reports, exports)
├─ reports/           # Report templates and final HTML reports
├─ scripts/           # Build / deployment scripts
├─ skills/            # Reusable skill modules
├─ src/               # Core source code (backend, services)
├─ templates/         # HTML templates for reports
└─ tests/             # Unit and integration tests
```

## Getting Started

1. Install Node.js (>=20).
2. Run `npm install` to install dependencies.
3. Use `npm run dev` (placeholder) to start development.

> No application code exists yet; this repository provides the scaffolding for future implementation.
